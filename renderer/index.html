          Calendar subscriptions

          Paste an ICS / webcal URL from any calendar provider (Google, Apple, Outlook…). Events appear in the Calendar drawer alongside scheduled Huddles. Refreshed every 15 minutes.

          Captions (offline)

          Live captions in calls run locally via whisper.cpp — audio never leaves the device. Bigger models are more accurate but take longer to download and use more memory at inference time. The "Active" model is used for all calls.

          Notifications

          Per-channel notification settings live on each channel — click the bell icon in the channel header to cycle between Default / All messages / Mute.

          Global notification preferences will land here as they're built.

          Density

          Tighten the layout for more content, or breathe for readability. Applies instantly.

          Accent color

          Tints active states and AI affordances. Doesn't change semantic colors (call / live / danger).
